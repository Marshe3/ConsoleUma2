# ConsoleUma

콘솔에서 진행하는 우마무스메 스타일의 C++ 육성 게임입니다. 캐릭터를 선택한 뒤 턴마다 훈련, 휴식, 레이스 출전을 선택하며 능력치와 컨디션을 관리합니다.

## main 기준 기능

- 캐릭터 선택: 아그네스 타키온과 맨하탄 카페 중 하나를 선택합니다.
- 턴 진행: 게임 날짜를 기준으로 매 행동 후 다음 턴으로 진행합니다.
- 훈련: 스피드, 스태미나, 파워, 근성, 지능 훈련으로 능력치를 성장시킵니다.
- 휴식: HP를 회복하고 다음 턴으로 넘어갑니다.
- 레이스: 현재 날짜에 출전 가능한 레이스를 선택해 참가합니다.
- 레이스 판정: 초반, 중반, 후반, 종반 4개 구간에서 능력치와 스태미나를 기반으로 순위를 계산합니다.
- NPC 상대: 플레이어 캐릭터와 NPC들이 함께 레이스에 참가합니다.

## feature/race-presentation 패치 내용

- 콘솔 UI를 넓은 레이아웃과 색상 출력 중심으로 개선했습니다.
- 캐릭터 화면에 각질과 고유기 정보를 표시합니다.
- 레이스 결과에 순간별 진행 장면(`RaceMoment`)과 주자 상태(`RaceRunnerState`)를 추가했습니다.
- 레이스 시뮬레이션에 각질(`RunningStyle`), 레인 선택, 마군 막힘, 외곽 주행, 고유기 발동을 반영했습니다.
- 레이스 중 트랙 형태의 진행 화면과 실황 코멘트를 출력합니다.
- NPC 능력치가 날짜 진행도에 따라 성장하도록 변경했습니다.
- 레이스 일정의 이름과 경기장 표기를 한국어로 정리했습니다.

## 빌드

Visual Studio 또는 Rider에서 `ConsoleUma.sln`을 열고 `Debug|x64` 또는 `Release|x64` 구성으로 빌드합니다.

프로젝트는 Windows 콘솔 API와 `_getch()`를 사용하므로 Windows 환경 실행을 기준으로 합니다.

## 조작

- `1`: 훈련
- `2`: 휴식
- `3`: 레이스
- `0`: 이전 메뉴로 돌아가기
- `q`: 게임 종료
