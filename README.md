# ConsoleUma

콘솔 환경에서 동작하는 우마무스메 스타일의 C++ 육성 시뮬레이션 게임입니다.  
캐릭터를 선택하고, 훈련·휴식·레이스를 반복하며 능력치와 컨디션을 관리해 최강의 우마무스메를 육성하세요.

---

## 플레이 화면

```
┌─────────────────────────────────────────────────────────────────┐
│  아그네스 타키온              쥬니어급 전반                        │
│  각질: 선행   고유기: U=ma2                                       │
│                                                                   │
│  스피드   72   스태미나  63   파워     68                         │
│  근성     62   지능     88   HP  75 / 100                        │
│                                                                   │
│  [1] 훈련   [2] 휴식   [3] 레이스   [Q] 종료                     │
└─────────────────────────────────────────────────────────────────┘
```

## 주요 기능

### 캐릭터 선택
| 캐릭터 | 각질 | 고유기 | 특징 |
|--------|------|--------|------|
| 아그네스 타키온 | 선행 | U=ma2 | 지능 특화, 고유기 발동률 우수 |
| 맨하탄 카페 | 차입 | 그대를 좇아서 | 스태미나·근성 특화, 후반 역전형 |

### 훈련
5가지 능력치를 강화할 수 있습니다. HP 소모가 있으며 HP가 낮을수록 실패 확률이 높아집니다.

| 훈련 종류 | 주 강화 능력치 |
|-----------|---------------|
| 스피드 훈련 | 스피드 |
| 스태미나 훈련 | 스태미나 |
| 파워 훈련 | 파워 |
| 근성 훈련 | 근성 |
| 지능 훈련 | 지능 |

### 레이스
- 현재 날짜에 개최되는 레이스만 출전 가능합니다.
- 레이스는 **초반 → 중반 → 후반 → 종반** 4개 구간으로 진행됩니다.
- NPC들과 함께 경쟁하며, NPC 능력치는 날짜 진행도에 따라 성장합니다.
- 각질(선행·차입 등), 레인 선택, 마군 막힘, 외곽 주행, 고유기 발동이 순위에 영향을 줍니다.
- 레이스 참가 시 HP가 25 소모됩니다.

### 날짜 시스템
- 게임 날짜는 **쥬니어급 전반**부터 시작합니다.
- 훈련·휴식·레이스를 선택할 때마다 1턴 진행됩니다.

## 클래스 구조

```
Character (능력치 기반)
└── Player (HP, 각질, 고유기)
    ├── AgnesTachyon
    └── ManhattanCafe

Race           - 레이스 시뮬레이션 및 결과 계산
RaceSchedule   - 날짜별 레이스 일정 관리
Training       - 훈련 실행 및 능력치 증가 계산
GameDate       - 게임 내 날짜(년/월/전후반) 관리
Npc            - 레이스 상대 NPC 생성
ConsoleRenderer - 콘솔 UI 출력 전담
```

## 빌드 방법

**요구 환경:** Windows, Visual Studio 2019 이상 (또는 JetBrains Rider)

1. `ConsoleUma.sln`을 Visual Studio 또는 Rider에서 엽니다.
2. 빌드 구성을 `Debug|x64` 또는 `Release|x64`로 선택합니다.
3. 빌드 후 생성된 실행 파일을 실행합니다.

> Windows 콘솔 API(`_getch()`, `SetConsoleOutputCP` 등)를 사용하므로 **Windows 전용**입니다.

---

## 조작 방법

| 키 | 동작 |
|----|------|
| `1` | 훈련 선택 |
| `2` | 휴식 (HP 회복: 30 / 50 / 70 중 랜덤) |
| `3` | 레이스 출전 |
| `0` | 이전 메뉴로 돌아가기 |
| `Q` | 게임 종료 |

## 기술 스택

- **언어:** C++17
- **빌드 시스템:** MSBuild (`.vcxproj`)
- **플랫폼:** Windows x64
- **외부 라이브러리:** 없음 (표준 라이브러리 + Windows API만 사용)
